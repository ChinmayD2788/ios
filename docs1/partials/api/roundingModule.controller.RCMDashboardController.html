RCMDashboardController
controller in module roundingModule

Description

  • Main Controller for RCM Dashboard Screen
  • Filter functionality is captured in roundingModule.controller:RCMDashboardFilterController
  • Sort functionality is captured in roundingModule.controller:RCMDashboardSortController
  • roundingModule.service:RCMDashboardService
  • VersionOne Requirements - Home Screen/Worklist - Patient Details - TK-19119
  • Supress Ability to switch roster - TK-19065
