PatientPathwaysTabController
controller in module roundingModule

Description

  • Controller for RCM Patient Pathways

Methods

  • bindPatientSurveyDetails()

    Bind Survey Details for selected patient.

  • getPatientSurveyDetails()

    Makes use of PatientPathwaysTabService GetPatientSurveyDetails function to load survey.

  • getSurveyDetails()

    Makes use of PatientPathwaysTabService GetSurveyDetails function to load survey.

  • onGetPatientSurveyDetailsRetrieved(result)

    • callBack for getPatientSurveyDetails service method
    Parameters
    ParamTypeDetails
    resultobject

    Patient Details

  • onGetSurveyDetailsRetrieved(result)

    callBack for getSurveyDetails service method.

    Parameters
    ParamTypeDetails
    resultobject

    Retrieving Patient Details for Pathways.

  • onGroupRadioChecked()

    Selecting options for pathways.

  • onSaveSurveyDetailsCompleted(result)

    1] callBack method for saveSurveyDetails. 2] Save survey details of the selected patient after completion.

    Parameters
    ParamTypeDetails
    resultobject

    Saved survey details of the patient.

  • onSwitchChange()

    Switch Change for Treatment Regimen Pathway.

  • show()

    Show survey based on the Pathway screen selected for RCM.

  • surveyCommentsBlur()

    Disable comments section in Pathway.

Properties

  • $scope.SurveyDetails

    • Load Survey for selected patient.

  • $scope.SelectedSurveyType

    • Selected Survey Type.

  • $scope.SurveyType

    • Type of Survey

  • $scope.Buttons

    • Edit, Save & Cancel button on Pathways screen.

  • $scope.IsSurveyCommentVisible

    • Visibility of Survey Comments.

  • $scope.IsSurveyDetailsCommentDisable

    • Value to decide whether Survey Comment Section should be disabled or enabled.

  • $scope.noActiveAdmissionRecord

    • Boolean value to check for admission record of a patient.

Events

  • EditPatientPathways()

    1] ng-click event for Editing pathways on view. 2] Edit selected pathway.

  • cancelPatientPathways()

    1] ng-click event for cancel Pathways on the view. 2] Revert changes made to the selected Pathway.

  • savePatientPathways()

    1] ng-click event called on click of Save button on Pathways Screen 2] Saves survey questions along with its response
