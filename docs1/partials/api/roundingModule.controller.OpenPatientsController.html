OpenPatientsController
controller in module roundingModule

Description

Controller for Open Patients Footer

Methods

  • showOpenPatients()

    • Loads the data grid of open patients
    • Sets the $scope.OpenPatientsModel.Total
    • Gets call when OpenPatientsController loads

Properties

  • $scope.OpenPatientsModel

    model of OpenPatientsController

  • $scope.OpenPatientsModel.Total

    property of $scope.OpenPatientsModel which is used to get total open patients

  • $scope.OpenPatientsModel.OpenPatients

    property of $scope.OpenPatientsModel which carries open patients data

  • $scope.OpenPatientsModel.GridOptions

    property of $scope.OpenPatientsModel which is used to display open patients in grid

  • $scope.OpenPatientsModel.OpenPatientsShow

    property of $scope.OpenPatientsModel which sets OpenPatientsShow flag

Events

  • PreSaveContactList

    • Global watch for PreSaveContactList

  • changeOpenPatientsShowFlag

    • Subscriber of changeOpenPatientsShowFlag event which will set boolean value for OpenPatientsShow

  • onOpenPatientSelected

    • k-on-tap event of kendo touch
    • Patient Selection Event from Opened Patients
    • Calls CommonFunctions.NavigatetoSelectedPatient

  • onOpenSelectedPatient

    • change event of grid item
    • Calls CommonFunctions.NavigatetoSelectedPatient

  • openPatientsShowClick

    • ng-click event of patient div
    • Handles open patients animation
