NewVapPlanController
controller in module roundingModule

Description

  • Controller for New Vap Plan Modal View
  • Child Controller for VAP (Vascular Access Plan) Screen

Methods

  • newPlanValidator()

    • Creates custom validator using kendo validator using different business rules for new plan modal view

Properties

  • $scope.model

    model of NewVapPlanController

  • $scope.model.DesiredPermanentAccess

    property of $scope.model used for Desired Permanent Access dropdown

  • $scope.model.VapSteps

    property of $scope.model used for Vap Steps List

  • $scope.model.SelectedStepOrder

    property of $scope.model used for Selected Step Order

  • $scope.model.Maxdate

    property of $scope.model used for Max Date

  • $scope.model.Mindate

    property of $scope.model used for Min Date

  • $scope.model.PlanStartDateConfig

    property of $scope.model used for Plan Start Date datepicker

  • $scope.model.PlanStartDate

    property of $scope.model used for Plan Start Date

  • $scope.model.AccessTypes

    property of $scope.model used for Access Type dropdown

Events

  • onAccessTypeChanged

    • ng-change event of access type change from new plan modal view
    • Calls VapService.GetVapStepDetails
    • Changes UI behaviour

  • onCancelNewVapPlanClick

    • k-on-tap event of Cancel Button of new plan modal view
    • Cancels Patient Vascular Access Plan
    • Closes new plan modal view
    • Changes UI behaviour

  • onOkNewVapPlanClick

    • k-on-tap event of Ok Button of new plan modal view
    • Saves Patient Vascular Access Plan
    • Calls VapService.SavePatientVascularAccess
    • Calls VapService.RefreshVap
    • Changes UI behaviour

  • onVapItemSelected

    • ng-click event of step selection from new plan modal view
    • Changes UI behaviour
