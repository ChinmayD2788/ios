StopVapPlanController
controller in module roundingModule

Description

  • Controller for Stop Vap Plan Modal View
  • Child Controller for VAP (Vascular Access Plan) Screen

Methods

  • stopPlanValidator()

    • Creates custom validator using kendo validator using diffrent business rules for stop vap plan modal view

Properties

  • $scope.model

    model of StopVapPlanController

  • $scope.model.PlanStopDate

    property of $scope.model used for VAP Stop date

  • $scope.model.StopReason

    property of $scope.model used for VAP Steps List

  • $scope.model.StopReasonOther

    property of $scope.model used for Selected Step Order

  • $scope.CopyVapStepDetails

    property of $scope used for local copy of VAP Step Details

  • $scope.ActivateVapStepDetails

    property of $scope used for Active Vap Step Detail

  • $scope.model.StopReasons

    property of $scope.model used for Stop Reason dropdown

Events

  • onCancelStopVapPlanClick

    • k-on-tap event of Cancel Button of stop vap plan modal view
    • Closes stop vap plan modal view
    • Changes UI behaviour

  • onStopVapPlanClick

    • k-on-tap event of Stop Button of stop vap plan modal view
    • Stops Current Vap Plan
    • Calls VapService.SavePatientVascularAccess
    • Calls VapService.RefreshVap
    • Changes UI behaviour
