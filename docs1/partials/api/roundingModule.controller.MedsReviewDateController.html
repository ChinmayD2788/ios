MedsReviewDateController
controller in module roundingModule

Description

  • Child controller of MedicationController
  • Controller for Medication Review Date Modal View

Properties

  • $scope.model

    model of MedsReviewDateController

  • $scope.model.MaxReviewDate

    property of $scope.model used for max review date

  • $scope.model.MinReviewDate

    property of $scope.model used for min review date

  • $scope.model.MedReviewDate

    property of $scope.model used for meds review date

  • $scope.model.PatientRefused

    property of $scope.model used for patient refused flag filter

Events

  • onCancelMedsReviewClick

    • k-on-tap event of Cancel Button of MedsReview Modal View
    • Closes modal view

  • onCreateMedsReviewClick

    • k-on-tap event of Save button of MedsReview modal view
    • Saves medication review date
    • Calls MedicationsService.GetIntMedicationSurvey
    • Calls MedicationsService.SaveSurveyDetails
    • Closes modal view
